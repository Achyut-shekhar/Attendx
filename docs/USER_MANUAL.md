# User Manual

## For Faculty

### Getting Started
1. **Login:** Use your registered faculty credentials.
2. **Dashboard:** You will see a list of your "Active Classes".
3. **Create Class:** Click the `+ Create Class` button to add a new course.

### Running a Session
1. **Start Session:**
   - Click the "Play" icon on a class card.
   - Choose "Start with Location" if you want to restrict students to the classroom radius (requires GPS).
   - A unique **6-character code** will generate.
2. **Display Code:**
   - Project the code on the screen.
   - Students will enter this code to join.
   - The screen will auto-refresh every **10 seconds** to show new attendees.
3. **End Session:**
   - Click "End Session" when class is over.
   - This prevents any further check-ins.
   - **Note:** Attendance cannot be changed by students after the session ends.

### Managing Attendance
- **Manual Mode:** You can click "View Details" -> "Manual Attendance" to tick checkboxes for specific students if they forgot their phone.
- **Reports:** Go to "Class Details" to export formatted Excel sheets for admins.

---

## For Students

### Joining a Class
1. **First Time Only:** Click "Join Class" and enter the code your professor gave you (e.g., `CS101`). You only do this once per semester.

### Marking Attendance
1. **In Class:** Open the app and find your class card.
2. **Enter Code:** Click "Enter Code" and type the 6-character session code on the projector.
3. **Location Check:** If enabled by the professor, you MUST allow browser location access.
   - *Tip:* Ensure your GPS is on before clicking submit.
4. **Success:** You will see a green "Attendance Marked" message. You are done!
   - *Note:* The button will lock ("Submitting...") to prevent accidental double-clicks.
